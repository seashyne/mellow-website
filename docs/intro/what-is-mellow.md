# What is Mellow

Mellow is a scripting language designed for:

- 🎮 Game Logic
- 🤖 AI Behaviors
- ⚙️ Automation
- 🛡️ Safe Sandbox execution

Key goals:

- Deterministic execution
- Simple Python-like syntax
- Built-in AI utilities
- Lightweight runtime
