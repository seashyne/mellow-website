# Syntax Guide

Mellow uses indentation-based syntax that should feel familiar if you have used Python-style scripting.

```mellow
let hp = 100

def add(a, b):
    return a + b

print(add(2, 3))

if hp < 10:
    print("low")
elif hp < 50:
    print("warning")
else:
    print("ok")
```

Use this page as a quick scan. For the stable 2.9 language contract, see the Core Profile page.
